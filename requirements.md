# SurakshaNet - AI crowd Disaster Prevention and Response Sytem

# Problem Statement
India experiences massive crowd gatherings like festivals, pilgrimages, railway stations, and rallies. Each year, the resulting crowding at these spots causes stampedes, resulting in the loss of precious lives. These events happen mostly because of the lack of real-time crowd monitoring, late emergency response, and the absence of early warning systems. 

SurakshaNet aims to prevent such tragedies by using AI-powered crowd analysis and predictive alerts.

## target Users
1. Police and crowd management authyorities
2. Event organizers
3. Disaster response teams
4. Smart city public safety departments

## Proposed Solution
SurakshaNet is an AI-based real-time crowd monitoring and disaster prediction platform that:

1. Detects and counts people in live video feeds
2. Measures crowd density in different zones
3. Identifies abnormal crowd movement or panic patterns
4. Predicts stampede risk before it happens
5. Sends  automatic alerts to authorities

## key Features 
1. Real-time crowd Detection and counting
2. Crowd Density Heatmap Visualization
3. Movement and Panic Anomaly Detection
4. Risk score prediction for stampede prevention
5. Automated Emergency alerts via sms/App
6. Dashboard for Live Monitoring

## AWS Resources Used
1. Amazon Rekognition - Crowd detection and video analytics
2. Amazon SageMaker - Training  risk prediction and anomaly models
3. AWS lambda - Serverless alert triggering 
4. Amazon SNS - SMS and emergency notifications
5. Amazon Quicksight - Crowd safety dashboard and reporting
6. Amazon EC2 - Deployment of AI inference pipeline

## Expected Impact
1. Prevent stampede disasters and save lives
2. Improve public safety in high-risk Zones
3. Provide scalable AI-based crowd control for Bharat 
4. Enable smart goverance and faster emergency response
